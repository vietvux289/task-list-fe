## 🚀 Getting Started

```bash
# 1. Clone the repository
git clone https://github.com/your-username/task-list-fe.git

# 2. Navigate to the project folder
cd task-list-fe

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev
```
